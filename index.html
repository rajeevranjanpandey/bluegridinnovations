Live Monitoring Made Easy

This ESP8266 + Arduino UNO-based system monitors water levels in real time and displays them via Wi-Fi on a second device. No LoRa or cloud dependency. 100% local.

Technology Stack

- Arduino Uno (Tank side controller)
- Ultrasonic Distance Sensor (DYP-A02YYUW)
- ESP8266 Wi-Fi Modules (x2)
- 0.96 inch OLED Display
- Laptop Wi-Fi Hotspot or Router
- Solar Power (Optional / 50W Panel Supported)
