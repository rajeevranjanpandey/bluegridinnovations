Wireless Water Tank Monitoring

GitHub Page: watertankmonitor

Project Overview

This project monitors the water level in a tank using an ultrasonic sensor and Arduino Uno. Data is transmitted wirelessly to a NodeMCU and displayed on an OLED screen. It uses WiFi (ESP8266) for real-time wireless communication.
