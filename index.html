Wireless Water Tank Level Monitoring System

Powered by Arduino Uno, ESP8266, Ultrasonic Sensor, and OLED Display

🔧 Project Overview

This project provides a reliable and wireless method to monitor water levels in a tank using Arduino Uno and ESP8266 modules. The system measures distance using an ultrasonic sensor, sends data over Wi-Fi, and displays it on an OLED screen.

📦 Components Used

  • Arduino Uno (Main Controller)
  • HC-SR04 Ultrasonic Sensor (Distance Measurement)
  • ESP8266 (Transmitter & Receiver Modules)
  • 0.96" OLED Display (Data Output)
  • Wi-Fi Router or ESP-based Wi-Fi bridge
  • 5V Power Source / 50W Solar Panel

📋 Algorithm


1. Start system and initialize components
2. Arduino Uno measures distance using HC-SR04
3. Distance sent over serial to Transmitter ESP8266
4. Transmitter ESP sends distance via UDP to Receiver ESP
5. Receiver ESP receives data and processes
6. OLED displays water level in Liters and Percentage
7. Repeat every 1 second
